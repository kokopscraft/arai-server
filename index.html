新井鯖について

新井建国鯖は20人ほど入れるレンタルサーバーのXServer Gamesを使って公開されているMinecraft統合版の建国サーバーです。

定期的な賭博イベント・PVPイベントや追加新要素があり、オリジナル要素を含んでいてとても楽しめます。

新井鯖のアドオン

新井鯖はさまざまなアドオンを組み合わせて独自の体験を提供しています。

  • Commander API
    コマンドの限界を突破する。コマンド勢に新たな力を。
    開発者: 𝒩𝒶𝓃ℴ・あるたか・wako
    ↗
  • 塩雪トランプアドオン
    52枚＋ジョーカー、ペルシャ絨毯青小・大、ペルシャ絨毯ピンク、透明額縁を追加。床にカーペットのように置くことができます。
    開発者: 雪塩 マイクラ
    ↗
  • Arai Server Add-on（オリジナル）
    新井鯖技術班のココが制作したオリジナル機能満載のアドオン。Java版の経済サーバーまんじゅうサーバーから着想を受けています。非公開・配布予定なし。
