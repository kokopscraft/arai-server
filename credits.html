使用中のアドオン

  • Commander API 2 [BP] GitHub
    開発者: 𝒩𝒶𝓃ℴ , あるたか
    コマンドの限界を突破する Commander API 本体。新井鯖では中核の基盤として採用しています。
  • Commander API Extension [BP] GitHub
    開発者: 𝒩𝒶𝓃ℴ , wako
    Commander API を拡張する追加モジュール。新井鯖では Commander API 2 と組み合わせて導入しています。
  • Commander API Screen [BP] GitHub
    開発者: 𝒩𝒶𝓃ℴ
    Commander API を使ったスクリーン表示機能を提供するアドオン。
  • Arai Server Add-on [RP][BP] Private
    開発者: ココ
    前提アドオン: Commander API 2 / Commander API Extension / Enchantments Level ZX
    Java版「まんじゅうサーバー」から着想を得た、新井鯖オリジナルの機能満載アドオン。未配布・配布予定なし。
  • Enchantments Level ZX (Template - Only Creative) [BP] MCPEDL
    開発者: masterkingTL ｜ @no-discord-user
    エンチャントレベルを大幅に拡張するテンプレートアドオン。長期間未更新ですが、ココが現バージョン対応に改造して採用しています。
  • Safe Storage [BP] MCPEDL
    開発者: Vechrozilator
    チェストに鍵をかけられる収納セキュリティアドオン。
  • Debug Stick [RP][BP] CurseForge
    開発者: vytdev
    デバッグスティックを追加するアドオン。
  • The Epic Knights Mod [RP][BP] MCPEDL
    開発者: The Mighty ChimMMMMmmmmm
    中世をテーマにした武器と防具を追加するアドオン。
  • WorldEdit: Bedrock Edition [RP][BP] MCPEDL
    開発者: SIsilicon
    Bedrock 版 WorldEdit。
